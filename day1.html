День 1: Старт и моделирование

Шаги на сегодня

  1. Запустите Blender и создайте новый проект: File → New → General.
  2. Выберите макет Layout.
  3. Освойте навигацию во вьюпорте (вращение, приближение, перемещение).
  4. Настройте единицы измерения и сохраните проект как lesson01.blend.
  5. Shift + A → Mesh → Plane — создайте основу ландшафта.
  6. Перейдите в Edit Mode (Tab), выделите всё (A) и выполните Subdivide (40–80 разрезов) для плотной сетки.
  7. Переключитесь в Sculpt Mode и начните формировать рельеф: холмы, впадины и широкое углубление (котлован) под будущую воду вокруг места для башни.
  8. Создайте основу башни-ротонды: цилиндр + колонны через модификаторы Array и Curve по кругу (кривая Bezier Circle); на скрине 1 виден уже собранный вид.
  9. Добавьте лестницу, отредактировав куб в Edit Mode (экструзия и подъём ступеней).

Горячие клавиши дня

  • Shift + A — добавить объект
  • Tab — переключение Object / Edit Mode
  • A — выделить всё
  • W → Subdivide — разбить плоскость
  • В Sculpt Mode: F — размер кисти, Smooth — сглаживание
