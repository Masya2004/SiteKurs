Чему научится человек с этого курса

  • Понимать интерфейс Blender и уверенно работать в основных режимах.
  • Моделировать окружение и архитектурные элементы с нуля.
  • Использовать PBR-текстуры, настраивать свет и камеру.
  • Делать чистый финальный рендер и оформлять проект для показа.

Шаги на сегодня

  1. Проверьте ОС: Windows / macOS / Linux (для Windows используем установщик `.exe`).
  2. Скачайте Blender с официального сайта: blender.org/download.
  3. Установите Blender (обычно оставьте параметры установки по умолчанию).
