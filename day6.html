Референс с обложки курса — башня у воды; кадр должен показывать и водную гладь, и архитектуру.

Шаги на сегодня

  1. Добавьте камеру: Shift + A → Камера (скрин 444).
  2. В аутлайнере выберите объект Camera (чтобы команда ниже знала, какую камеру двигать).
  3. Орбитой мыши настройте свободный вид в 3D-вьюпорте: так, как хотите видеть картинку в итоге — башня, вода, горизонт.
  4. Встаньте в нужный вид и нажмите Ctrl + Alt + Numpad 0 (ноль на цифровом блоке): активная камера встанет в текущий вид — позиция и угол совпадут с тем, что вы настроили глазами.
  5. Проверка: Numpad 0 — посмотреть сцену глазами камеры. Если нет NumPad: меню Вид → Выравнять вид → Активную камеру по виду для шага с выравниванием (или назначьте сочетание в настройках ввода).
  6. При необходимости чуть подвиньте камеру в виде Numpad 0 и снова примените Ctrl + Alt + Numpad 0, пока кадр не устроит.
  7. Правило третей: сместите башню от центра кадра, оставьте баланс неба и воды.
  8. В свойствах камеры задайте фокусное расстояние 35–50 мм (эквивалент). Размытие фона (Depth of Field) по желанию.
  9. Output Properties → разрешение, например 1920 × 1080.
  10. Сохраните lesson06_camera.blend.

Скрин: добавление камеры

Горячие клавиши этого дня

  • Shift + A — добавить камеру.
  • Ctrl + Alt + Numpad 0 — активная камера = текущий вид.
  • Numpad 0 — переключиться на вид из камеры.
